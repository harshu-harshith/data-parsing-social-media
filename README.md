# Social Media Data Parsing and Analysis Tool

## 📌 Project Overview
This project is a web-based tool developed to extract, parse and analyze user data exported from social media platforms such as Instagram. The system allows investigators or analysts to upload social media data archives and view structured information including messages, login details, media files and account activity.

The tool helps in organizing large amounts of digital evidence and reduces manual effort in analyzing social media data.

## ▶️ How to Run the Project
1. Download or clone this repository
2. Open the project folder
3. Run the application using Live Server or any local server
4. Upload a social media data export ZIP file
5. Use filter options to explore extracted information
